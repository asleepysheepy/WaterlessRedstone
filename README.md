# Waterless Redstone
A Minecraft Bukkit/Spigot Plugin that makes redstone and rails unaffected by water

Currently makes the following blocks unaffected by water

-Redstone Dust

-Redstone Repeaters

-Redstone Comparator

-Redstone Torch

-Tripwire Hook

-Tripwire String

-Stone Button

-Wooden Button

-Lever

-Rail

-Powered Rail

-Activator Rail

-Detector Rail
